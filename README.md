# Healthcare Operations & Patient Flow Analytics Dashboard

## Project Overview

This project is an end-to-end Healthcare Analytics Dashboard developed using Python, SQL, Power BI, and Streamlit. The dashboard analyzes patient admission trends, operational efficiency, wait times, departmental performance, and patient satisfaction metrics to generate actionable healthcare insights.

The objective of this project is to help healthcare organizations monitor patient flow, identify operational bottlenecks, and improve service efficiency through data-driven decision-making.

## Key Business Insights

- General Practice handled the highest patient volume.
- Higher patient wait times negatively impacted satisfaction scores.
- Peak patient admissions occurred during operationally busy hours.
- Certain departments showed possible operational bottlenecks.
