# Healthcare Operations & Patient Flow Analytics Dashboard

## Project Overview

The Healthcare Operations & Patient Flow Analytics Dashboard is an end-to-end data analytics project designed to analyze hospital operational performance, patient admission trends, wait times, department efficiency, and patient satisfaction metrics.

This project was developed using Python, SQL, Power BI, and Streamlit to demonstrate the complete analytics workflow including data cleaning, exploratory data analysis, KPI generation, dashboard visualization, and cloud deployment.

The dashboard helps healthcare organizations monitor patient flow, identify operational bottlenecks, and support data-driven decision-making.

---

# Problem Statement

Healthcare organizations often struggle with:

* High patient wait times
* Operational bottlenecks
* Uneven department workload distribution
* Difficulty monitoring patient satisfaction trends
* Lack of centralized analytical dashboards

This project aims to solve these challenges by building an interactive analytics dashboard for operational monitoring and healthcare insights generation.
